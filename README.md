# Air Drawing with Hand Tracking

Проект для рисования в воздухе с помощью камеры и руки.

## Возможности

- Распознавание руки через MediaPipe
- Рисование указательным пальцем
- Поддержка левой и правой руки
- Сглаживание линии
- Очистка холста (клавиша C)

## Запуск

```bash
pip install -r requirements.txt
python main.py
